Roof Joint Type (API 650 §5.10.2)

A frangible joint is standard for atmospheric tanks — the roof lifts before the shell separates in an overpressure event. Non-frangible requires P/V relief vents (excluded from this estimate).

Step 3 — Scope & Options

Select painting, accessories, fire protection, freight, erection method, and review the nozzle schedule.

Internal Coating

Paint height — select full tank height or specify a partial level:

External Coating

Applied separately to shell (vertical) and roof (horizontal/sloped — higher rate). Aluminium geodesic dome is excluded from external painting (vendor finish).

Floating Roof Coating

Top deck = weather-exposed face. Underside = product-contact face. Both can be coated independently.

Rim Seal System

Rim seal closes the gap between the floating roof edge and the shell. Primary + secondary is increasingly required by environmental regulations.

Shell Access
ℹ

Shell height is between 4–6 m. Select preferred access method.

Roof perimeter walkway (handrail + grating) is included for all fixed-roof and EFR tanks. Rolling ladder included automatically for EFR.

Stored Fluid Classification (SS 532)

Determines whether a fixed fire protection system is required and what type. Piping scope: top-of-shell ring header, vertical supply pipe, and flanged tie-in at tank shell bottom.

🚫 Class 0 (LPG/LNG) is out of scope for API 650 atmospheric tanks. Please consult a specialist engineer for cryogenic or pressurised storage.
